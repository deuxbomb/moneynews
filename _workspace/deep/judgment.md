## 환각 발견 (3건)

1. **[테슬라 섹션] FSD 누적 마일리지 구식 데이터**
   - 본문: "FSD 누적 주행 거리가 84억 마일(8.4 billion miles)입니다. 웨이모의 완전 비지도 주행 누적인 2억 마일의 42배"
   - 문제: Tesla가 2026년 5월 4일(기사 작성 1일 전) X에서 공식 발표 — 10억 마일(10B) 돌파. 42배 → 50배로 재계산 필요.
   - 처리: 10B로 업데이트 + Musk가 '10B = 비지도 준비 임계점'으로 발언한 사실 추가. 내러티브가 오히려 강화됨.
   - 출처: Electrek (2026.05.03), Tesla X 공식 계정

2. **[투자 섹션] Morgan Stanley 등급 오표기**
   - 본문: "모건스탠리 Hold 등급, 목표주가 $415"
   - 문제: Morgan Stanley의 공식 표기는 "Equal Weight(중립)". $415 목표주가는 정확.
   - 처리: "Equal Weight(중립) 등급"으로 수정.
   - 출처: TipRanks, MarketScreener, TheStreet

3. **[죽스 섹션] "30만 명에게 무료 시승" 미검증 수치**
   - 본문: "지금까지 30만 명에게 무료 시승을 제공하며 데이터와 브랜드 인지도를 쌓는 중이에요."
   - 문제: 출처 미확인. 검색에서 해당 수치를 뒷받침하는 공식 발표 없음.
   - 처리: 수치 삭제. "2025년 9월 라스베이거스 스트립 일대에서 공개 서비스를 시작하여 브랜드 인지도를 쌓아왔어요."로 대체.

## 개선점 (6건)

1. **[테슬라 섹션] FSD 10B 마일 업데이트**
   - 8.4B → 10B 업데이트. Musk의 "10B = 완전 비지도 준비 임계점" 발언 연결. 단순 수치 교정을 넘어 서사를 강화할 기회.

2. **[투자 섹션] Morgan Stanley 등급 정확화**
   - "Hold" → "Equal Weight(중립)". 사소해 보이지만 투자 관련 글에서 등급 표기 오류는 치명적.

3. **[죽스 섹션] 타임라인 명확화**
   - 죽스의 직접 서비스(2025.09 라스베이거스 공개)와 Uber 앱 통합(2026년 목표)이 혼재되어 있음. 독자 혼란 방지를 위해 분리 서술.

4. **[웨이모 섹션] 센서 절감 수치 정밀화**
   - "센서 비용을 40%까지 낮췄지만" → 실제: 센서 수 42% 감소, 단위 원가 50% 이상 절감. 구체성 부족.

5. **[비교 섹션] 시장 전망 단정 표현 완화**
   - "$55억 → $2,751억"은 단일 보도자료 기준. 복수 기관 범위($1,500억~$2,800억)로 표기하거나 "추정치에 따라 차이가 있지만"으로 완화.

6. **[결론] 마지막 문장 강화**
