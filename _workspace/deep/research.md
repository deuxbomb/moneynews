### B. FAA 타입 인증 진행 현황

Joby의 FAA 타입 인증은 5단계 프로세스:
1. G-1(인증 기반 확립) → 2. 설계 검토 → 3. 시험 계획 수립 → 4. SR3 감사·시험 실행 → **5단계: TIA(최종)**

- **2026년 3월 말**: FAA가 4단계 완료 공식 확인, SR3 감사 통과
- **현재 5단계(TIA = Type Inspection Authorization)**: FAA 시험 조종사가 직접 Joby 기체 비행
- TIA용 FAA 적합 기체 4대 모두 생산 중
- 5단계 완료 = 미국 최초 eVTOL 상업 여객 서비스 인가

출처: Aircraft Insider, eVTOL.Travel, FAA/Joby Q1 어닝콜

## 반론/리스크

1. **FAA 5단계 기간 불확실성**: Boeing 787 인증이 예상 대비 수년 지연된 선례. TIA 과정에서 소프트웨어·설계 미비 발견 시 재지연 가능 — 2026년 내 인증 실패 시 주가 급락 리스크
2. **현금 소진**: 분기 $110~150M 소진 → 상업화 전 추가 자금조달 불가피, EPS 희석 반복
3. **Archer 선점 리스크**: FAA 자체 4단계 중 3단계 완료 "최초" 타이틀 + 두바이·미국 동시 공략. Archer가 Joby보다 먼저 미국 인증 받을 경우 시장 선점 가능성
4. **EHang 저비용 위협**: 조종사 없는 자율 에어택시 → 근본적으로 다른 운항 비용 구조. 규제 허용 확대 시 Joby 비즈니스모델 위협
5. **Supernal 붕괴 교훈**: 현대차 $6B 투자 의도에도 80% 해고 → 잘 자금화된 회사도 인증 지연+시장 불확실성에 무너질 수 있음
6. **가격 경쟁력**: 에어택시 수익성 유지 위한 좌석당 요금 vs 지상 우버풀/자율주행 (Waymo ~$2~3/마일). 공항 연결 외 대중화 경로 제한적
7. **시장 수요 불확실성**: 항속 150마일 eVTOL이 실제 도심 사용 패턴과 맞는가? 럭셔리 틈새 vs 대중 교통수단 사이 포지셔닝 모호
