# 레퍼런스 기사 분석 — SpaceX IPO 딥다이브 (2026-05-24)

## 레퍼런스 기사

### 1. "SpaceX's biggest IPO risks are also its bull case" [Yahoo Finance](https://finance.yahoo.com/markets/article/spacexs-biggest-ipo-risks-are-also-its-bull-case-173659815.html)
- **구조**: 역설(paradox) 구조 — "리스크가 곧 불케이스다"라는 반직관적 명제로 시작 → 각 리스크가 동시에 왜 투자 근거가 되는지 설명 → 결론
- **문체**: 날카롭고 냉소적, 월스트리트 관행을 비틀어 독자 시선을 끄는 방식. 짧은 문장과 대비적 구조 사용
- **도입부**: 높은 밸류에이션 배수가 리스크면서 동시에 가장 매력적인 투자 근거라는 반전 설정
- **숫자 활용**: "Facebook도 16개월 소화 기간" 등 역사적 비교로 숫자에 맥락 부여
- **배울 점**: 반전 구조(risks = bull case)를 섹션 4(반대 시각)에서 각 리스크가 동시에 기회인 이유 양면 분석에 활용

### 2. "Inside SpaceX's S-1: Three Companies. One Profit. $1.75 Trillion." [Satnews](https://satnews.com/2026/05/21/inside-spacexs-s-1-three-companies-one-profit-1-75-trillion/)
- **구조**: "세 회사, 하나의 이익"이라는 핵심 명제 → 사업부 분해(Connectivity/Space/AI) → 각 부문 별 숫자 → 투자자가 실제 사는 게 무엇인지
- **문체**: 전문적이고 데이터 중심. 표 형태 재무 데이터 + 서술 분석 병용
- **도입부**: S-1 핵심 역설 — 세 사업부 중 하나만 흑자, 나머지 둘이 그것을 소진
- **숫자 활용**: 부문별 매출/EBITDA/Operating Income을 병렬 제시 후 통합 해석
- **배울 점**: "세 엔진" 프레임워크를 우리 글에서도 사용. Q1 2026 부문별 실적 표로 시각화 적용

### 3. "SpaceX IPO Is No Longer Reserved for the Ultra Rich" [TipRanks](https://www.tipranks.com/news/spacex-ipo-is-no-longer-reserved-for-the-ultra-rich-heres-how-retail-investors-can-buy)
- **구조**: 독자(소매 투자자)의 페인 포인트 → 해결책(Robinhood/Fidelity 접근) → 구체적 방법 → 주의사항
- **문체**: 친근하고 실용적, "당신도 살 수 있다"는 임파워먼트 톤
- **도입부**: 기존 엘리트 전용 IPO의 장벽이 무너졌다는 뉴스 앵커
- **숫자 활용**: 락업 기간, 할당 비율(30%) 등 실행 가능 정보에 초점
- **배울 점**: 시사점 섹션 마지막에 한국 투자자 접근 방법(미래에셋, Tiger ETF)을 실용적으로 안내하는 구조 차용
