## 문체 지침 (여러 레퍼런스에서 공통 추출)

1. **도입**: 구체적 사건/순간으로 시작. "2025년 7월 18일, 법에 서명한 후..." 식의 영화적 도입
2. **섹션 전환**: 각 인물 소개 전 짧은 "상황 설정" 1문단 삽입
3. **숫자 맥락화**: "$3.6B"가 아닌 "$3.6B — 한국 씨티은행 연간 예금 규모"식의 비교 맥락
4. **인용 활용**: 직접 인용 → "왜 이 말이 중요한가" 설명 → 숫자 연결의 3단 패턴
5. **교차 분석**: 찬성/반대를 대립 구도로 놓되, "둘 다 옳다" 또는 "두 세계가 충돌한다"는 제3의 시각으로 마무리
6. **투자 시사점**: 단순 종목 나열이 아닌 "어떤 조건이 충족될 때 사야 하는가" 체크리스트 형태
